# AI-Nexus

A modern, responsive AI technology agency website developed as part of the **INNOVIAST Week 1 – Frontend Experience Engineering Internship**.

**Objective:**  
Create a polished static website for an AI technology agency that looks professional across desktop, tablet, and mobile devices.

**Author:** Milhan Musharaf

---

# Overview

AI-Nexus is a responsive landing page for a fictional AI product studio that develops intelligent automation solutions, AI-powered products, and modern web experiences. The website is designed with a clean, professional interface and provides an engaging user experience across all screen sizes.

---

# Features

- Responsive Navigation Bar
- Hero Section with Primary Call-to-Action
- About Section
- AI Products & Services Section
- Webinar Section
- Contact Form
- Footer with Navigation
- Mobile-Friendly Design
- Contact Form Validation
- Smooth Responsive Layout

# Design System

The website uses a custom design system with reusable CSS variables.

| Token | Value | Purpose |
|-------|-------|---------|
| --ink | #14131a | Background |
| --paper | #ece6d9 | Light Sections |
| --amber | #e8a33d | Primary Button |
| --teal | #3e6259 | Secondary Accent |
| --rust | #c44536 | Highlight Color |

Typography uses:

- Fraunces
- Inter
- Space Mono

Spacing follows an 8px design system for consistency.

# Running Locally

No installation is required.

### Option 1

Simply open:

```
index.html
```

in your web browser.

### Option 2 (VS Code)

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click **index.html**.
4. Click **Open with Live Server**.

---

# Responsive Design

The website is fully responsive and optimized for:

- Desktop (1440px+)
- Laptop (1024px)
- Tablet (768px)
- Mobile (390px)
- Small Mobile (360px)

Features include:

- No horizontal scrolling
- Responsive navigation
- Flexible layouts
- Adaptive typography
- Mobile-friendly spacing
