# AI Nexus

## Overview

AI Nexus is a modern responsive landing page developed as part of the InnoViast Week 1 Frontend Experience Engineering Internship. The project represents an AI technology agency offering intelligent automation, web development, mobile applications, and cloud-based solutions.

---

## Features

- Responsive Design
- Hero Section
- About Section
- Services
- Portfolio
- Testimonials
- FAQ
- Contact Form
- Sticky Navigation
- Smooth Scrolling

---

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Setup

1. Download the project.
2. Open the folder in VS Code.
3. Install the Live Server extension.
4. Open index.html.
5. Click Go Live.
